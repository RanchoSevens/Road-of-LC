#### 求数列的所有子集

```java
class Solution {
    public List<List<Integer>> subsets(int[] nums) {
        List<List<Integer>> res = new ArrayList<List<Integer>>();
        res.add(new ArrayList<>());
        for(int num : nums){
            int n = res.size();

            for(int i = 0; i < n; ++i){
                List<Integer> tmp= new ArrayList(res.get(i));
                tmp.add(num);
                res.add(tmp);
                // System.out.println("---");
                // System.out.println(res);
                // System.out.println("+++");
            }
        }
        return res;
    }
}
```

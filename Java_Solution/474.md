474. Ones and Zeroes (Medium)
- 题目描述
  - 给定 m 个数字 0 和 n 个数字 1，以及一些由 0-1 构成的字符串，求利用这些数字最多可以构 成多少个给定的字符串，字符串只可以构成一次。

```java
/*
 * Input: strs = ["10","0001","111001","1","0"], m = 5, n = 3
 * Output: 4
 * Explanation: The largest subset with at most 5 0's and 3 1's is {"10", "0001", "1", "0"}, so the answer is 4.
 * Other valid but smaller subsets include {"0001", "1"} and {"10", "1", "0"}.
 * {"111001"} is an invalid subset because it contains 4 1's, greater than the maximum of 3.
 * 思路：0-1背包问题，限制条件：i 序列，j 0的数量 状态转移方程：dpm[j] = Math.max(dpm[j - 1], dpm[j - count[i][0]] + 1)
 *      注意：dpm和dpn有相关关系，所以必须放在一处讨论 -> 三维空间 降到二维
 */

class Solution {
    public static int findMaxForm(String[] strs, int m, int n) {
        int num = strs.length;
        int[][] count = new int[num + 1][2]; // 0和1的数量
        for (int i = 0; i < num; i++) count[i + 1] = zeroandone(strs[i]);

        int[][] dp = new int[m + 1][n + 1];

        for (int i = 1; i <= num; i++) {
            for (int j = m; j >= count[i][0]; j--) {
                for (int k = n; k >= count[i][1]; k--) {
                    dp[j][k] = Math.max(dp[j][k], dp[j - count[i][0]][k - count[i][1]] + 1);
                }
            }
        }
        return dp[m][n];
    }

    private static int[] zeroandone(String str) {
        int[] cn = new int[2];
        char[] chs = str.toCharArray();
        for (char c : chs) cn[c - '0']++;
        return cn;
    }

    public static void main(String[] args) {
        String[] strs = {"10","0001","111001","1","0"};
        System.out.println(findMaxForm(strs, 4, 3));
    }
}
```
